Possibilidades de pseudo-classes em links

link
configuração original do link
visited
configuração para o link já visitado
hover
configuração para o link quando o cursor passar por cima
active
configuração para o link ao ser clicado
focus
configuração para o link em foco

Possibilidades de pseudo-classes em listas

first-child
configurar o primeiro filho do elemento
last-child
configurar o último filho do elemento
